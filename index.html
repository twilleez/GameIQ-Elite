Elite Onboarding Tutorial

This replaces the basic welcome dialog with a focused walkthrough for first-time coaches.

Start on the coach dashboard for game context and AI reads before jumping into tracking.
2. Quick Game Mode
Toggle Quick Game Mode when you need less clutter and faster logging on the sideline.
3. Tap the court
Shot chart taps auto-detect 2PT vs 3PT and update player shooting instantly.
4. AI insights
Insights are explainable. They react to FG%, turnovers, scoring load, and foul pressure.
5. Save every game
Saved games feed season charts, player comparison, and roster-level reporting.
